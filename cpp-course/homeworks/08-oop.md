# Homework 08: OOP

1. Create a `Person` class with name and age.
2. Add getters and setters with validation.
3. Create a `Car` class with constructor and destructor.
4. Create a base class `Shape` with virtual `area()`.
5. Implement `Circle` and `Rectangle` derived classes.
6. Store shapes in a vector of base pointers and print areas.
7. Bonus: Demonstrate object slicing and explain how to avoid it.
