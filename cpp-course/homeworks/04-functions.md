# Homework 04: Functions

1. Write a function to return the square of a number.
2. Write a function to check if a number is prime.
3. Write a function to reverse an integer.
4. Write a recursive function for Fibonacci.
5. Write a function to compute power using fast exponentiation.
6. Write a function to swap two integers using references.
7. Bonus: Write a function that returns the median of three numbers.
