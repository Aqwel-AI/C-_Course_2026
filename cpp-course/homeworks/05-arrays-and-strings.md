# Homework 05: Arrays and Strings

1. Read 10 integers into an array and print the maximum.
2. Reverse an array in place.
3. Count vowels in a string.
4. Check if a string is a palindrome.
5. Find the second largest element in an array.
6. Rotate an array by K positions.
7. Bonus: Implement basic string compression (e.g., "aaabb" -> "a3b2").
