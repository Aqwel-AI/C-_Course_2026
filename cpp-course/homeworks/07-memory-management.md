# Homework 07: Memory Management

1. Allocate and delete a single integer on the heap.
2. Allocate and delete an array of 10 integers.
3. Convert a raw pointer example to `std::unique_ptr`.
4. Implement a class that manages a dynamic array (Rule of Three).
5. Demonstrate reference counting with `shared_ptr`.
6. Identify and fix a memory leak in a short snippet.
7. Bonus: Implement a small RAII wrapper for a file handle.
