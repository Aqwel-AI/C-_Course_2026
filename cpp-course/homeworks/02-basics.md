# Homework 02: C++ Basics

1. Declare variables of types `int`, `double`, `char`, and `bool`.
2. Read two integers and output their sum, product, and average.
3. Read a full name using `std::getline` and print it.
4. Convert a temperature from Celsius to Fahrenheit.
5. Print a 5x5 multiplication table.
6. Print the ASCII code of a character input by the user.
7. Bonus: Write a program that swaps two values without a temp variable.
